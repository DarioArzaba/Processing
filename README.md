### Installation

1. Download [Processing](https://processing.org/tutorials/) (Java Based) IDE (Currently 4.2)
2. Clone the repo
   ```sh
   git clone https://github.com/DarioArzaba/Processing.git
   cd Processing
   ```
3. Open, edit and run any of the `pde` files provided.
